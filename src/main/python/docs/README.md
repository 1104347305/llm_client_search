## 最短上手路径

如果时间很紧，按这个路径走：

1. 看 `models/schemas.py` 和 [04-Schema与配置说明.md](./04-Schema与配置说明.md)，先理解最终输出结构。
2. 看 `steps/query_router.py` 和 [02-代码框架与解析链路.md](./02-代码框架与解析链路.md)，理解解析流程。
3. 新增字段从 `config/field_definitions_args.yaml` 开始。
4. 新增规则从 `config/enhanced_rules_args.yaml` 开始。
5. 用 `/api/v1/fields/retrieve` 检查字段召回。
6. 用 `/api/v1/client_search_query_parse_no_encipher` 检查最终解析结果。
7. 修改后执行配置热更新，再补测试。
