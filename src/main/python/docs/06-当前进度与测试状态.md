## 变更记录

### 2026-05-14：e生保生效客户解析口径修正

- 问题：`e生保生效客户` 容易被泛化解析为“存在保单生效日”，即 `polNoInfo.poleffdate EXISTS`。
- 口径：这里的“生效客户/生效中客户”表示保单状态有效，不表示有保单生效日。
- 实现：将口语表达 `生效`、`生效中` 合并到 L2 规则 `保单状态-有效汇总`，并让“投保险种简称/名称 + 保单状态”组合规则引用该汇总规则。
- 预期输出：`polNoInfo.plancodeinfo.abbrname MATCH e生保` + `polNoInfo.polStatus CONTAINS [交费有效, 自垫交清, 交清, 减额交清, 免交, 自垫有效]`。
- 回归：新增 `test_l2_maps_eshenbao_effective_customer_to_policy_status_not_effective_date`，明确断言不输出 `polNoInfo.poleffdate`。

## 主要失败原因

### 1. 测试仍引用旧路径

多个测试还在引用迁移前的模块路径：

```python
from core.level2_enhanced_matcher import Level2EnhancedMatcher
from core.query_router import QueryRouter
from core.field_registry import FieldRegistry
from services.search_service import SearchService
```

但当前源码路径已经迁移为：

```python
from src.main.python.steps.level2_enhanced_matcher import Level2EnhancedMatcher
from src.main.python.steps.query_router import QueryRouter
from src.main.python.steps.field_registry import FieldRegistry
from src.main.python.services.search_service import SearchService
```

### 2. 部分测试文件存在语法错误

例如：

```text
src/main/python/tests/test_level1.py
src/main/python/tests/test_logic_operators.py
```

错误形态：

```python
from sys.path.insert(...)
```

这不是合法 Python 语法，应改为：

```python
import sys
sys.path.insert(0, "...")
```

或直接移除旧路径注入，统一使用新包路径。

### 3. 部分测试引用不存在模块

例如：

```python
from core.level2_template_matcher import Level2TemplateMatcher
```

当前项目中没有对应模块，需要确认是历史遗留测试还是模块迁移遗漏。

## 建议修复顺序

1. 统一测试 import 路径。
2. 修复语法错误测试文件。
3. 删除或重写引用不存在模块的测试。
4. 对需要 ES/LLM/下游搜索 API 的测试增加 mock。

## 当前风险点

### 仓库处于迁移中的 dirty 状态

`git status` 显示大量旧根目录文件删除、新 `src/main/python/` 文件新增/移动。后续合并前需要确认这是预期迁移，而不是误删。

### L3 配置存在但主链路未启用

配置中：

```yaml
ENABLE_L3: true
```

但 `QueryRouter.route_with_peeling()` 中 L3 相关调用被注释。对外说明时应避免宣称 L3 已实际生效。

### 搜索接口代码存在但未注册路由

`SearchService` 与 `SearchAPIClient` 存在，但当前 FastAPI router 没有暴露自然语言搜索或结构化搜索 route。

### 配置文件包含明文密钥

当前环境 YAML 中存在 `LLM_API_KEY`。建议迁移到环境变量或密钥管理系统。

### FastAPI startup API 有弃用告警

当前使用：

```python
@app.on_event("startup")
```

pytest 输出 FastAPI deprecation warning。后续可以迁移到 lifespan。
