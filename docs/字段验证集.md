# 字段验证集

说明：
- 本验证集基于当前 `config/field_definitions.yaml` 中 `is_supported != false` 的字段生成。
- 当前可支持字段共 `42` 个。
- 验证集分为三层：
  - 单字段标准问法
  - 单字段口语问法
  - 多字段组合问法（含复杂问题）
- 可直接用于 Level2、RAG、Level4 和摘要链路回归。

## 七、建议使用方式

- Level2 回归：优先跑 `S`、`M023`、`M025`、`M039`、`M081-M099`

- L4 回归：优先跑 `M031-M050`
- 摘要回归：优先跑带日期、家庭成员、多字段 OR/AND 的问题
