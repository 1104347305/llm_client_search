# 新查询规则分析

## 需要添加的规则类型

### 1. 客户价值+温度组合规则 ✅ (部分已有)
- A类冷却客户
- A类中温客户
- A类低温名单
- 黄金VIP低温e生保有哪些人
- 40岁左右高价值低温有哪些人
- 中高温A类客户

**现状**: 已有客户价值、客户温度单独规则，需要添加组合规则

### 2. 明天生日 ✅ (需添加)
- 明天生日的人

**现状**: 已有"下个月生日"、"本月生日"，需添加"明天"、"今天"、"后天"

### 3. 总保费字段 ❌ (数据库无此字段)
- 总保费90万以上

**问题**: 数据库只有 `annual_premium` (年缴保费)，没有 `total_premium` (总保费)
**建议**: 使用 `annual_premium` 替代，或需要数据库添加此字段

### 4. 联系记录字段 ❌ (数据库无此字段)
- 长期没联系保额10万以上都有谁
- 帮我查30岁左右最近没联系寿险名单

**问题**: 数据库没有 `last_contact_date` 或 `contact_records` 字段
**建议**: 需要数据库添加此字段

### 5. 身份证有效期 ❌ (数据库无此字段)
- 身份证有效期快到期的客户

**问题**: 数据库只有 `certificates.id_number`，没有有效期字段
**建议**: 需要数据库添加 `certificates.expiry_date` 字段

### 6. 客户添加时间 ✅ (已有字段)
- 把近一个月新添加的客户查询出来
- 最近新增客户
- 上个月添加的客户

**现状**: 有 `customer_added_date` 字段，需添加相对时间规则

### 7. 寿险VIP+产品组合 ✅ (需添加组合规则)
- 哪些是黄金VIP意外险的人
- 帮我找VIP保费一万以上名单
- A1意外险客户

### 8. 年龄+产品组合 ✅ (需添加)
- 查30岁养老险的人
- 帮我查40岁左右车险客户
- 给我看看40岁低温分红险客户

### 9. 未购买/未配置保险 ✅ (需完善)
- 每购意外险的客户 (应为"未购")
- 找45岁以上没有配置保险的

### 10. 保单周年日 ✅ (已有字段)
- 哪些客户下个月到保单周年日了

**现状**: 有 `policy_anniversary` 字段，需添加规则
